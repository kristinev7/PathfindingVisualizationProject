# Pathfinding Visualizer

A react web app to visualize pathfinding algorithms.

[Check it out!](https://g-r3.github.io/Pathfinding-Visualizer/)

## Algorithms

-   Dijkstra
-   A\* (A star)
-   Breadth First Search
-   Depth First Search
